## Fluxo para PROFESSOR

1. Abra Meus Projetos → Dashboard.
2. Gere um projeto a partir do template (ou crie/edite um projeto existente).
3. Assine digitalmente o projeto na própria interface.
4. Submeta para aprovação administrativa.
5. Confirme que o status do projeto mudou para "SUBMITTED".

---

## Fluxo para ADMINISTRADOR

1. Abra Projetos → Gerenciar Projetos.
2. Localize um projeto com status "SUBMITTED".
3. Visualize o PDF do projeto (quando disponível) para conferência.
4. Aprove o projeto.
5. Confirme que o status mudou para "APPROVED".

---

## Fluxo para ALUNO

1. Abra Monitoria → Vagas Disponíveis.
2. Selecione uma única vaga de interesse (uma disciplina/projeto).
3. Clique em Inscrição em Monitoria e confirme a inscrição.
4. Verifique sua inscrição em Monitoria → Resultados das Seleções (sem depender de e-mail).
