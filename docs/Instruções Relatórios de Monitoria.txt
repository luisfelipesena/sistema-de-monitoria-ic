INSTRUÇÕES PARA SUBMISSÃO DE RELATÓRIOS DE MONITORIA 2025.1


Professor responsável pelo projeto de monitoria da disciplina:
        
1. Faça uma cópia do Anexo III, preencha o relatório final do projeto de monitoria e salve uma cópia assinada no formato PDF na pasta Relatórios.
2. Edite a Planilha de Certificados com os dados do relatório final do projeto de monitoria (Planilha planilha_certificados_2025_1 - Aba Relatórios Disciplinas) e no nome da disciplina (Coluna A)  insira o link para o arquivo PDF do passo anterior.


Professor orientador que assina o relatório do monitor:


1. Faça uma cópia do Anexo II, preencha o relatório de atividades do monitor e salve uma cópia assinada no formato PDF na pasta Relatórios.
2. Edite a Planilha de Certificados com os dados do monitor (Planilha planilha_certificados_2025_1 - Aba Relatórios Monitores) e no nome do monitor (Coluna A)  insira o link para o arquivo PDF do passo anterior.
3. Podem acessar as planilhas dos bolsistas e voluntários do semestre 2025.1 para conferirem os dados dos seus monitores.


* Você pode acessar a pasta de Relatórios de Monitoria 2024.1 ou Relatórios de Monitoria 2024.2 dos semestres passados e aproveitar os textos de preenchimento dos relatórios naqueles semestres.
