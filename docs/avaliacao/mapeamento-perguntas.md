# Mapeamento: Formulário Original → Entrevista Qualitativa

## Comparação de Abordagens

| Aspecto | Original (Likert) | Nova (Qualitativa) |
|---------|-------------------|---------------------|
| N participantes | Múltiplos | 2 |
| Validade estatística | Necessária | Não aplicável |
| Tipo de dado | Numérico (1-5) | Textual (insights) |
| Profundidade | Superficial | Rica em detalhes |
| Análise | Médias/desvios | Análise temática |

## Perguntas Originais → Reformulação

### Q1: Notificação automática após importação
**Original (Likert 1-5):**
> "A notificação automática após importação do planejamento é útil"

**Nova (Entrevista - Admin):**
> "Após importar a planilha, você conseguiu verificar quais professores foram notificados? O feedback do sistema foi suficiente?"

**Observação no Walkthrough:**
- Verificar se admin encontra histórico de notificações
- Observar reação ao ver status de envio

## Estrutura de Análise para TCC

### Categorias de Análise

1. **Usabilidade**
   - Facilidade de navegação
   - Clareza das interfaces
   - Feedback do sistema
   - Erros e recuperação

2. **Utilidade**
   - Atende às necessidades reais
   - Economia de tempo
   - Automação efetiva

3. **Confiabilidade**
   - Confiança na assinatura digital
   - Segurança percebida
   - Integridade dos dados

4. **Adoção**
   - Barreiras identificadas
   - Pré-requisitos para uso real
   - Necessidade de treinamento

5. **Sugestões**
   - Melhorias propostas
   - Funcionalidades desejadas
   - Ajustes de interface

## Justificativa Metodológica para TCC

> "Devido ao número limitado de participantes (N=2), optou-se por uma abordagem qualitativa exploratória em vez de avaliação quantitativa. Estudos qualitativos com amostras pequenas são apropriados para identificar problemas de usabilidade e obter insights profundos sobre a experiência do usuário (Nielsen, 1994; Lazar et al., 2017). A metodologia combinou walkthrough guiado com protocolo think-aloud e entrevista semiestruturada, permitindo triangulação de dados observacionais e autorreportados."
